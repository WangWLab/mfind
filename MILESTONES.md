## 当前进展详情

### 本次更新 (2026-03-23)

#### 新增功能

1. **FSEvents 监控 (M6)** ✅
   - 实现 macOS FSEvents watcher (`fsevents.rs`)
   - 基于轮询的实时监控（生产环境可替换为原生 FSEvents API）
   - 事件批处理和去重机制
   - 支持目录递归监控

2. **索引增量更新 (M7)** ✅
   - `IndexEngine::update()` 完整实现
   - 支持 Create/Delete/Modify/Rename 事件处理
   - inode 映射和元数据缓存同步
   - FST 索引动态重建

### 已推送功能

```bash
# 搜索文件
mfind search '*.rs'           # 通配符搜索
mfind search 'Cargo'          # 前缀搜索
mfind search -e rs            # 扩展名过滤
mfind search -r '.*\.rs$'     # 正则搜索

# 构建索引
mfind index build <path>      # 构建指定路径索引

# 输出格式
mfind search '*.rs' -o json   # JSON 输出
mfind search '*.rs' -o list   # 列表输出 (默认)
```
