# gen-test-data - 高性能测试数据生成器

支持生成**千万级**文件，优化了磁盘空间占用和目录分布。

## 特性

- **高并发**: 默认 500 并发，快速生成
- **智能目录分布**: 自动创建足够目录，确保单目录不超过 `max_files_per_dir` 设定值
- **最小化模式**: 大幅减少磁盘占用（每文件仅~4 字节）
- **进度显示**: 实时显示创建进度
- **空间检查**: 自动检查磁盘可用空间

## 使用示例

### 生成 100 万文件（最小化模式，推荐）

```bash
cd scripts/gen-test-data

./target/release/gen-test-data \
  -c 1000000 \
  --minimal \
  --max-files-per-dir 1000 \
  -o ./test_data_1m
```

**实际测试结果**:
- 文件数：1,000,000
- 目录数：1,000
- 每目录文件数：1,000
- 总大小：3.8 GB
- 生成速度：~7,000 文件/秒

### 生成 1000 万文件

```bash
./target/release/gen-test-data \
  -c 10000000 \
  --minimal \
  --max-files-per-dir 1000 \
  -o ./test_data_10m
```

**预计资源占用**:
- 目录数：约 10,000 个
- 每目录文件：~1000 个
- 磁盘空间：~38 GB
- 预计耗时：~25 分钟

### 生成 10 万文件（快速测试）

```bash
./target/release/gen-test-data \
  -c 100000 \
  --minimal \
  -o ./test_data_100k
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-c, --count` | 文件总数 | 1000 |
| `--concurrency` | 并发度 | 500 |
| `--max-files-per-dir` | 每目录最大文件数 | 1000 |
| `--minimal` | 最小化文件内容 | false |
| `-o, --output` | 输出目录 | ./test_data |
| `--for-integration` | 集成测试模式 | false |

## 性能参考

| 文件数 | 模式 | 预计耗时 | 磁盘占用 |
|--------|------|----------|----------|
| 10 万 | minimal | ~15 秒 | ~380 MB |
| 100 万 | minimal | ~2.5 分钟 | ~3.8 GB |
| 1000 万 | minimal | ~25 分钟 | ~38 GB |

## 注意事项

1. **磁盘空间**: 生成前先用小数量测试，确认空间足够
2. **清理文件**: 测试完成后用 `rm -rf` 删除测试目录
3. **并发度**: 过高并发可能导致系统负载过高，默认 500 已优化
4. **文件描述符**: 程序已优化避免超过系统限制
